SpaceCaptain
NFC Writer Setup // ESP32
Schritt 01
ESP32 per USB verbinden

ESP32 per USB an diesen Computer anschliessen. Chrome oder Edge verwenden (Web Serial API erforderlich).

Schritt 02
Firmware installieren

Auf "Installieren" klicken, den richtigen COM-Port auswählen und den Flash-Vorgang bestätigen.

Schritt 03
WLAN konfigurieren

Nach dem Flash startet der ESP32 neu und öffnet einen WLAN-Access-Point:

SSID
SpaceCaptain-NFC

Mit dem Smartphone oder Laptop mit diesem WLAN verbinden. Es öffnet sich automatisch ein Konfigurationsportal — dort SSID und Passwort des lokalen WLANs eingeben. Der ESP32 speichert die Zugangsdaten und verbindet sich künftig automatisch.

Schritt 04
IP-Adresse eintragen

Nach erfolgreicher WLAN-Verbindung zeigt der ESP32 seine IP-Adresse im Serial Monitor. Diese URL in den SpaceCaptain-Einstellungen unter NFC-Writer URL eintragen:

Beispiel
http://192.168.1.42
Hinweis: Erneutes Flashen setzt die WLAN-Einstellungen nicht zurück — die gespeicherten Zugangsdaten bleiben erhalten. Zum Zurücksetzen den BOOT-Button beim Start gedrückt halten.
