Kiosk Setup

Raspberry Pi als Display-Terminal einrichten

1
Raspberry Pi OS (Desktop) installieren

Raspberry Pi Imager herunterladen und Raspberry Pi OS (Desktop) auf eine SD-Karte schreiben. SSH und WLAN können direkt im Imager konfiguriert werden.

2
Terminal öffnen und Befehl ausführen

Terminal-App öffnen und folgenden Befehl einfügen und ausführen:

Lade...

Das Skript installiert Chromium, deaktiviert den Bildschirmschoner und richtet den Kiosk-Autostart ein.

3
Neustart

Nach Abschluss des Skripts den Pi neu starten. Chromium öffnet sich automatisch im Vollbild und zeigt die Display-Seite.

Hinweis: Das Skript läuft nur auf Raspberry Pi OS (Debian-basiert). Chromium wird über apt installiert. Für Updates einfach den Befehl erneut ausführen.
